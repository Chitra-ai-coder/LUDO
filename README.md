# 🎲 CLASSIC LUDO | Premium 3D Edition

A modern, high-end interpretation of the classic Ludo board game. Built with a focus on **Spatial UI**, **Cinematic Typography**, and a seamless responsive experience across Mobile, Tablet, and Desktop.

![Ludo Preview](https://img.shields.io/badge/UI-Premium%203D-blueviolet?style=for-the-badge)
![Platform](https://img.shields.io/badge/Platform-Web%20|%20Mobile%20|%20PC-orange?style=for-the-badge)

---

## ✨ Key Features

* **💎 Spatial 3D Design:** Floating pill-shaped top bar and tactile buttons with realistic depth and glassmorphism effects.
* **🎨 Dynamic Dice Engine:** A glowing centerpiece that physically changes color and cast-shadows based on the active player.
* **📱 Universal Responsiveness:** Smart viewport scaling ensures the board looks massive on PC but fits perfectly on mobile screens.
* **🎭 Cinematic Experience:** Optimized with custom Google Fonts (Outfit/Bebas Neue) for a high-end gaming aesthetic.
* **⚡ Mathematical Precision:** A custom grid-based movement engine that ensures tokens never misalign.

---

## 🚀 Technologies Used

* **HTML5:** Semantic structure for game screens.
* **CSS3:** Advanced Grid layouts, `clamp()` responsive math, and 3D transform effects.
* **JavaScript (ES6):** Core game logic, turn management, and collision detection.
* **Lucide Icons:** Clean, consistent iconography.
* **Google Fonts:** Modern geometric typography.

---

## 🕹️ How to Play

1.  **Launch:** Choose between **2-Player (1v1)** or **4-Player (Chaos)** modes on the immersive landing page.
2.  **Roll:** Click the dynamic, glowing dice to generate a number.
3.  **Move:** Click your tokens to move them across the board. 
4.  **Capture:** Land on an opponent's token to send them back to base (excluding safe star zones!).
5.  **Win:** Be the first to get all 4 tokens into the center home!

---

## 🛠️ Installation & Setup

Since this is a client-side web application, no complex installation is required:

1.  Clone this repository or download the source code.
2.  Open `index.html` in any modern web browser (Chrome, Safari, Edge).
3.  Enjoy the game!

---

## 🎨 Customization

To change the vibe of the game, you can modify the following in the code:

* **Dice Size:** Edit the `.dice-container` class in CSS.
* **Fonts:** Swap the Google Fonts link in the `<head>` to change the cinematic style.
* **Colors:** Update the `:root` variables to change player themes.
